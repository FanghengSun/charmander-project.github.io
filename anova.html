ANOVA testing for climate variables

Question we are answering:

Which climate variable has the largest effect on wild fire?

Methods

The statistical test we used to answer this question is One-way ANOVA. The null hypothesis is that the mean of each variable in days with fire and without fire is the same. By conducting ANOVA tests for all climate variables with respect to fire status(fire/no fire), the following resulted tables were generated. If there is a significant difference between fire days and no fire days(the null hypothesis is rejected) for a certain variable, it can be inferred that the specific climate variable has no significant effect on wild fire.

p-value > 0.05 , we fail to reject the null hypothesis, and conclude that there is no significant difference between FFMC index means in fire days and in no fire days.
